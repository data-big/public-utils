
# 常用开发工具类

> 基于Java实现。

### 使用方式

* 各项目在pom.xml中增加以下内容：

```java
<project ...>
        ...
        
        <!-- 依赖包 -->
        <dependency>
            <groupId>info.soft</groupId>
            <artifactId>public-utils</artifactId>
            <version>1.0.0</version>
        </dependency>
</project>
```

### 注意事项

> ProgramDriver目前针对RESTlet启动的接口服务不支持，会导致接口启动不成功。
